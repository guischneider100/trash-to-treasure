# TrashToTreasure

A community-powered app that lets users share and find free street items, promoting sustainability through reuse and waste reduction.

# Status

⚠️ Project is still in development! Bad functionalities and bugs can be anywhere.

# Technologies

- Backend: Spring Boot, Java, PostgreSQL
- Frontend: React Native
- Authentication: JWT

# Features

- Post items found on the street with location
- Browse items nearby
- Mark items as collected
- Item lifecycle: available → collected → removed after set time
- Upload images

# Roadmap

- Implement JWT authentication
- Notifications system
- Map-based browsing
- Improve UI/UX design
